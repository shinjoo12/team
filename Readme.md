# 안녕하세요

1.git init : 저장소를 선언한다.
2.git add : 깃의 스테이징 영역에 파일을 추가한다.
3.git commit : 현재 스테이징의 내용을 하나의 스냅샷으로 저장한다.
4.git push origin main : 원본 레파지토리의 메인 브랜치에 현재 저장된 내용을 업로드한다.
